# MLSys-Papers

每日 arXiv 大模型系统方向论文简报，自动搜索、整理、推送。

## 方向

- 📦 **Serving / 推理系统** — vLLM, 推理引擎, 服务化部署
- 🔄 **分布式训练** — FSDP, 模型并行, 通信优化
- 🚀 **推理加速** — 投机解码, Attention 优化
- 💾 **显存 / KV Cache 优化** — 显存管理, Cache 压缩
- ⚙️ **编译 / 量化** — Triton, MLIR, 权重量化
- 🏗️ **AI 基础设施** — GPU 调度, 集群, 资源管理

## 格式

每篇论文包含：标题、链接、作者、完整英文摘要 + 中文翻译。

## 自动更新

每日北京时间 14:00 自动搜索 arXiv 并推送。
