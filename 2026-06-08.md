### 1. Terastal: Layer-Variant-based Scheduling for Real-Time Multi-DNN Workloads on Heterogeneous Accelerators
- **ID:** 2606.06818v1
- **作者:** Sing-Yao Wu, Fengshuo Song, Eli Bozorgzadeh
- **链接:** https://arxiv.org/abs/2606.06818

**英文摘要:** Heterogeneous DNN accelerators improve soft real-time multi-DNN execution by mapping each layer to its preferred accelerator to reduce latency. However, under skewed workloads, large layer-latency differences across accelerators limit scheduling flexibility and increase deadline misses. To address this challenge, we introduce layer variants, customized layer implementations that reduce latency gaps on non-preferred accelerators. We then present Terastal, a soft real-time framework for layer-variant design and scheduling on heterogeneous DNN accelerators. Terastal combines offline heterogeneity-aware virtual budget assignment and layer-variant design, and online scheduling to jointly optimize accelerator mapping and variant selection under timing and accuracy constraints. Experimental results show that Terastal reduces deadline miss rate per model by 40.58%, 30.53%, and 36.27% compared with FCFS, EDF, and DREAM, respectively, while incurring only 2.24% average normalized accuracy loss across models with variants.

**中文翻译:** 异构 DNN 加速器通过将每一层映射到其偏好的加速器来改善软实时多 DNN 执行的延迟。然而，在偏斜的工作负载下，不同加速器之间较大的层延迟差异限制了调度灵活性并增加了截止时间错过率。为解决这一问题，本文引入了层变体（layer variants），即定制的层实现，用于减少非偏好加速器上的延迟差距。我们提出 Terastal，一个用于异构 DNN 加速器上层变体设计和调度的软实时框架。Terastal 结合了离线异构感知虚拟预算分配和层变体设计，以及在线调度，在时序和精度约束下联合优化加速器映射和变体选择。实验结果表明，与 FCFS、EDF 和 DREAM 相比，Terastal 将每模型的截止时间错过率分别降低了 40.58%、30.53% 和 36.27%，同时仅带来 2.24% 的平均归一化精度损失。

### 2. SCALE: Scalable Cross-Attention Learning with Extrapolation for Agentic Workflow Scheduling
- **ID:** 2606.06820v1
- **作者:** Zhifei Xu, Jierui Lan, Zixuan Liang, Aiji Liang, Jinxi He
- **链接:** https://arxiv.org/abs/2606.06820

**英文摘要:** Agentic Large Language Model (LLM) systems decompose complex tasks into workflow Directed Acyclic Graphs (DAGs) whose primitives must be scheduled on heterogeneous clusters. Existing deep reinforcement learning (DRL) schedulers are tied to a fixed cluster size and require retraining whenever the number of servers changes. We propose SCALE (Scalable Cross-Attention Learning with Extrapolation), a DRL scheduler that generalizes to unseen cluster scales without fine-tuning. SCALE employs a cross-attention pointer network where task features query against server features, so the architecture accepts any number of servers by construction. We observe, however, that permutation-invariant architecture alone does not guarantee good performance at new scales - the attention feature undergoes distribution shift as the server count grows. To counter this, we introduce Structured Representation Regularization (SRR): a decorrelation loss combined with a KL penalty toward the standard normal, which keeps feature statistics stable regardless of input size. Trained on 16 nodes and tested directly on 32 and 48 nodes, SCALE reduces average response time by 8.9% at N=48 relative to the same architecture without SRR, confirming that explicit regularization is necessary to close the scale-generalization gap.

**中文翻译:** 大语言模型智能体系统将复杂任务分解为工作流有向无环图，其基本单元需要在异构集群上调度。现有深度强化学习调度器绑定到固定集群规模，每当服务器数量变化时都需要重新训练。我们提出 SCALE，一种无需微调即可泛化到未见集群规模的 DRL 调度器。SCALE 采用交叉注意力指针网络，任务特征查询服务器特征，因此架构天然支持任意数量服务器。然而，仅靠置换不变架构并不能保证在新规模下的良好性能——注意力特征随服务器数量增长而发生分布偏移。为此，我们引入结构化表示正则化（SRR）：一种去相关损失加上朝向标准正态分布的 KL 惩罚，使特征统计量不受输入大小影响。在 16 节点上训练，直接测试于 32 和 48 节点，SCALE 在 N=48 时相比无 SRR 的架构将平均响应时间降低了 8.9%，证实了显式正则化对弥合规模泛化差距的必要性。

### 3. Clairvoyant: Predictive SJF Scheduling to Mitigate Head-of-Line Blocking in Serial LLM Backends
- **ID:** 2606.07248v1
- **作者:** Aravind Sundaresan
- **链接:** https://arxiv.org/abs/2606.07248

**英文摘要:** Serial LLM inference backends -- such as Ollama -- process requests one at a time under FCFS admission, causing Head-of-Line Blocking (HOLB) under mixed workloads at high utilisation: short factual queries can be delayed by minutes behind long generation jobs. While cloud-scale deployments mitigate HOLB via continuous batching (vLLM, Orca), these solutions require tens of GB of VRAM for concurrent KV-caches -- infeasible for memory-constrained edge and local deployments that rely on serial request dispatch. We present Clairvoyant, a drop-in sidecar proxy for any serial OpenAI-compatible backend (e.g., Ollama, llama.cpp). Clairvoyant predicts response length from 19 lightweight lexical features via an ONNX-exported XGBoost classifier, achieving 0.029ms per-request latency (four orders of magnitude below typical generation time). End-to-end GPU benchmarks on an RTX 4090 show 70-76% P50 latency reduction for short requests under maximum queue pressure (100 concurrent requests) and 17% under steady-state Poisson arrivals (ρ=0.74). Clairvoyant is open-source and requires no modifications to the inference backend.

**中文翻译:** 串行 LLM 推理后端（如 Ollama）在 FCFS 接收策略下一次处理一个请求，在高利用率混合工作负载下导致队头阻塞（HOLB）：短查询可能因排在长生成任务后面被延迟数分钟。虽然云规模部署通过连续批处理（vLLM、Orca）缓解了 HOLB，但这些方案需要数十 GB VRAM 来容纳并发 KV 缓存——对于内存受限的边缘和本地部署不可行。我们提出 Clairvoyant，一个适用于任何串行 OpenAI 兼容后端（如 Ollama、llama.cpp）的即插即用 sidecar 代理。Clairvoyant 通过 ONNX 导出的 XGBoost 分类器，从 19 个轻量级词法特征预测响应长度，每请求延迟仅 0.029ms（比典型生成时间低四个数量级）。在 RTX 4090 上的端到端 GPU 基准测试显示，在最大队列压力（100 并发请求）下短请求的 P50 延迟降低 70-76%，在稳态泊松到达（ρ=0.74）下降低 17%。Clairvoyant 已开源，无需修改推理后端。

### 4. Breaking the Ice: Analyzing Cold Start Latency in vLLM
- **ID:** 2606.07362v1
- **作者:** Huzaifa Shaaban Kabakibo, Animesh Trivedi, Lin Wang
- **链接:** https://arxiv.org/abs/2606.07362

**英文摘要:** As scalable inference services become popular, the cold start latency of an inference engine becomes important. Today, vLLM has evolved into the de facto inference engine of choice for many inference workloads. Although popular, due to its complexity and rapid evolution, there has not been a systematic study of its startup latency. With major architectural innovations such as the V1 API and the introduction of torch.compile, this paper presents the first detailed performance characterization of vLLM startup latency. We break down the startup process into six foundational steps and demonstrate that it is predominantly CPU bound. Each step exhibits consistent and interpretable scaling trends with respect to model-level and system-level parameters, enabling fine-grained attribution of latency sources. Building on these insights, we develop a lightweight analytical model that accurately predicts vLLM startup latency for a given hardware configuration, providing actionable guidance for resource planning in large-scale inference environments.

**中文翻译:** 随着可扩展推理服务的普及，推理引擎的冷启动延迟变得至关重要。如今 vLLM 已成为许多推理工作负载的事实标准推理引擎。尽管广受欢迎，但由于其复杂性和快速演进，尚未有对其启动延迟的系统性研究。本文首次对 vLLM 启动延迟进行了详细的性能表征，涵盖了 V1 API 和 torch.compile 等重大架构创新。我们将启动过程分解为六个基本步骤，并证明其主要受 CPU 限制。每个步骤在模型级和系统级参数上表现出一致且可解释的扩展趋势，从而能够实现细粒度的延迟来源归因。基于这些发现，我们开发了一个轻量级分析模型，可准确预测给定硬件配置下 vLLM 的启动延迟，为大规模推理环境中的资源规划提供可操作的指导。

### 5. SigmaScale: LLM Compression with SVD-based Low-Rank Decomposition and Learned Scaling Matrices
- **ID:** 2606.07098v1
- **作者:** Ernests Lavrinovics, Marco Letizia, Roy Janco, Shai Segal, Johannes Bjerva
- **链接:** https://arxiv.org/abs/2606.07098

**英文摘要:** We present SigmaScale, a method for learning auxiliary scaling matrices S to aid truncated Singular Value Decomposition (SVD) based Large Language Model (LLM) compression. Instead of deriving scaling matrices analytically, SigmaScale optimizes two sets of vectors that define diagonal row and column scaling transformations under an activation-aware compression loss. We show that learned scaling lowers the effective intrinsic rank of weight matrices, as reflected by reductions in effective-rank entropy, and that this reduction is strongly correlated with compression loss. Experiments on Llama 3.1 8B Instruct and Qwen3-8B show that SigmaScale is competitive with closely related state-of-the-art SVD-based compression methods across perplexity and zero-shot benchmarks.

**中文翻译:** 本文提出 SigmaScale，一种学习辅助缩放矩阵 S 以改善基于截断奇异值分解（SVD）的大语言模型压缩的方法。不同于解析推导缩放矩阵，SigmaScale 在激活感知的压缩损失下优化两组定义对角行和列缩放变换的向量。我们证明学习到的缩放降低了权重矩阵的有效本征秩（反映在有效秩熵的降低上），且这种降低与压缩损失强相关。在 Llama 3.1 8B Instruct 和 Qwen3-8B 上的实验表明，SigmaScale 在困惑度和零样本基准上与密切相关的 SVD 压缩方法竞争力相当。

### 6. Sparsely Gated Tiny Linear Experts
- **ID:** 2606.07414v1
- **作者:** Simon Schug
- **链接:** https://arxiv.org/abs/2606.07414

**英文摘要:** Sparsity allows scaling model parameters without proportionally increasing computational cost. While mixture of experts (MoE) models are made increasingly sparse, individual experts typically remain large and dense. Here, we demonstrate that further increasing sparsity by shrinking each expert to consist of a single neuron and selecting a tiny fraction of many available neurons can improve compute efficiency and interpretability. Counterintuitively, the key to achieving both is removing the nonlinearity typically applied to the experts, resulting in a network of sparsely gated linear neurons (sgatlin). In an isoflop comparison, we find that replacing all transformer feedforward layers with sgatlin improves perplexity in language models across different compute budgets. At the same time, the sparsity and linearity of the resulting feedforward circuits present new opportunities for model interpretability.

**中文翻译:** 稀疏性允许在不按比例增加计算成本的情况下扩展模型参数。虽然混合专家（MoE）模型越来越稀疏，但单个专家通常仍然大而密集。本文证明，通过将每个专家缩小到单个神经元并选择众多可用神经元中的极小一部分来进一步提高稀疏性，可以提升计算效率和可解释性。反直觉的是，实现两者的关键是移除通常应用于专家的非线性激活，从而产生稀疏门控线性神经元（sgatlin）网络。在等 FLOPs 比较中，我们发现用 sgatlin 替换所有 Transformer 前馈层可以在不同的计算预算下改善语言模型的困惑度。同时，所得前馈电路的稀疏性和线性为模型可解释性提供了新的可能性。

### 7. Towards Tight Bounds for Streaming Attention
- **ID:** 2606.07205v1
- **作者:** Justin Y. Chen, Ying Feng, Piotr Indyk, Michael Kapralov, Ekaterina Kochetkova
- **链接:** https://arxiv.org/abs/2606.07205

**英文摘要:** The attention mechanism is a cornerstone of modern transformer architectures. However, its expressive power comes at the cost of quadratic runtime and linear space usage. The problem of implementing a transformer in limited space, known as KV cache compression, has received much interest over the past few years, spurring the development of powerful heuristics. In this work, we revisit the streaming attention approximation problem and provide nearly tight bounds on its space complexity. On the algorithmic side, we achieve the result through a tight interplay between three distinct methods for kernel density estimation: discrepancy-based coreset constructions, the polynomial method, and space partitioning. On the lower bound side, our main technical contribution is a new technique for using the INDEX problem with a large amount of side information.

**中文翻译:** 注意力机制是现代 Transformer 架构的基石。然而，其表达能力以二次运行时和线性空间使用为代价。在有限空间中实现 Transformer 的问题（称为 KV 缓存压缩）在过去几年中受到了广泛关注，推动了强大启发式算法的发展。本文重新审视流式注意力近似问题，并为其空间复杂度提供了近乎紧致的上下界。在算法方面，我们通过三种核密度估计方法（基于差异的 coreset 构造、多项式方法和空间划分）之间的紧致协同来实现结果。在下界方面，我们的主要技术贡献是使用带有大量辅助信息的 INDEX 问题的新技术。这项工作为 KV 缓存压缩的极限能力提供了理论基础。

### 8. OffQ: Taming Structured Outliers in LLM Quantization by Offsetting
- **ID:** 2606.07116v1
- **作者:** Haoqi Wang, Lorenz K. Mueller, Jiawei Zhuang, Mathieu Salzmann, Lukas Cavigelli
- **链接:** https://arxiv.org/abs/2606.07116

**英文摘要:** Low-bit quantization has been widely adopted to accelerate the inference of large language models (LLMs) by significantly reducing computational cost and memory usage. However, activation outliers pose a major challenge to effective quantization, often leading to notable performance degradation. In this paper, we introduce OffQ, a method designed to mitigate activation outliers in low-bit quantization through a novel offsetting mechanism. Specifically, OffQ first identifies a low-dimensional outlier subspace in the activations using a proposed top-1 PCA, and then concentrates high-magnitude activations into 1 channel via rotation. OffQ then absorbs this concentrated outlier channel by converting its magnitude into a shared offset, thereby reducing the standard deviation of the activations. This offsetting strategy enables effective W4A4KV4 quantization of LLMs using deployment-friendly uniform-grid and uniform-precision quantization. Extensive experiments across diverse LLM architectures and benchmarks demonstrate that OffQ outperforms state-of-the-art baselines, consistently improving model accuracy while preserving low-bit efficiency.

**中文翻译:** 低位量化已被广泛用于加速大语言模型推理，显著降低计算成本和内存使用。然而，激活值异常点对有效量化构成重大挑战，常常导致显著的性能下降。本文提出 OffQ，一种通过新颖的偏移机制来减轻低位量化中激活值异常点影响的方法。具体而言，OffQ 首先通过提出的 top-1 PCA 识别激活值中的低维异常点子空间，然后通过旋转将高幅值激活集中到 1 个通道中。OffQ 随后将此集中的异常通道通过将其幅度转换为共享偏移来吸收，从而降低激活值的标准差。这种偏移策略使得 LLM 能够使用部署友好的均匀网格和均匀精度量化实现 W4A4KV4 量化。跨多种 LLM 架构和基准的大量实验表明，OffQ 优于现有最先进方法，在保持低位效率的同时稳定提升模型精度。

### 9. Reversible Foundations: Training a 120B Sparse MoE through State-Preserving Scaling
- **ID:** 2606.07404v1
- **作者:** Rohan Shravan
- **链接:** https://arxiv.org/abs/2606.07404

**英文摘要:** This paper reports on training a hundred-billion-parameter sparse mixture of experts on a single eight-GPU node, end to end. LightningLM 0.1V is a recurrence-backbone language model family grown in four stages from a small dense seed, through a 5B and a 9B mixture of experts, to a 120B model with 460 routed experts under top-12 routing. Active parameters rise monotonically from 1.78B at the dense seed to 5.93B at 120B (about 5% of the 118.67B stored). The full lineage runs on single nodes, the larger stages at 8K context. This is a systems and experience report organized around three disciplines. Reversibility: a reversible recurrence stack reconstructs activations in the backward pass instead of storing them, holding activation memory flat as the model grows. State-preserving growth: each expansion is given as a reproducible principle paired with the failure that results from getting it wrong. Single-node economics: the 120B trains through TQP, a strategy of quantized base expert weights and trained low-rank adapters that carries optimizer state on 2.26B adapter parameters rather than 100B+ resident in routed experts, cutting expert-path optimizer state by ~45x.

**中文翻译:** 本文报告了在单个八 GPU 节点上端到端训练千亿参数稀疏混合专家模型的经验。LightningLM 0.1V 是一个基于循环主干的语言模型家族，经过四个阶段从小型稠密种子模型逐步增长：经过 5B 和 9B 的 MoE 阶段，最终达到 120B 参数规模（460 个路由专家，top-12 路由）。活跃参数从稠密种子的 1.78B 单调增加到 120B 时的 5.93B（约占存储的 118.67B 的 5%）。这是一篇围绕三个核心原则的系统实现与经验报告：可逆性——可逆循环栈在后向传播时重建激活而非存储它们，使激活内存随模型增长保持平坦；状态保持增长——每次扩展都作为可复现原则给出，伴以错误操作会导致的失败；单节点经济性——通过 TQP 策略使用量化基础专家权重和训练的低秩适配器，将优化器状态承载在 2.26B 适配器参数上而非路由专家中的 100B+，将专家路径优化器状态削减约 45 倍。

### 10. Closed-Form Spectral Regularization for Multi-Task Model Merging (SWUDI)
- **ID:** 2606.07289v1
- **作者:** Yongxian Wei, Runxi Cheng, Xingxuan Zhang, Li Shen, Chun Yuan
- **链接:** https://arxiv.org/abs/2606.07289

**英文摘要:** Model merging combines several independently fine-tuned experts into a single multi-task model without any training data, reducing the storage, serving, and decentralized-development costs of large foundation models. In this work, we formalize multi-task model merging as a noisy linear inverse problem and propose a spectral filtering estimator parameterized by a per-direction filter. We instantiate this estimator with SWUDI, a closed-form method that combines a soft exponential filter with a hard top-K truncation that suppresses noise-amplifying small-eigenvalue directions. Across four general benchmarks and a multimodal merging benchmark, our proposed spectral solvers match or outperform state-of-the-art merging methods while reducing wall-clock time by 28-72x and peak GPU memory by up to 50%.

**中文翻译:** 模型合并将多个独立微调的专家模型组合成一个多任务模型，无需任何训练数据，从而降低大规模基础模型的存储、服务和去中心化开发成本。本文将多任务模型合并形式化为一个带噪线性逆问题，并提出一种由每方向滤波器参数化的谱滤波估计器。我们以 SWUDI 实例化该估计器，这是一种结合软指数滤波器与硬 top-K 截断的闭式方法，可抑制放大噪声的小特征值方向。在四个通用基准和一个多模态合并基准上，我们提出的谱求解器匹配或超越了最先进的合并方法，同时将运行时间减少 28-72 倍，峰值 GPU 内存降低高达 50%。

### 11. ANNS-AMP: Accelerating Approximate Nearest Neighbor Search via Adaptive Mixed-Precision Computing
- **ID:** 2606.07156v1
- **作者:** Mingkai Chen, Cheng Liu, Shengwen Liang, Lei Zhang, Xiaowei Li
- **链接:** https://arxiv.org/abs/2606.07156

**英文摘要:** Approximate nearest neighbor search (ANNS) is a critical kernel in modern applications such as LLM and recommendation systems. However, its efficiency is fundamentally limited by the need to compute distances between a query and a massive number of high-dimensional vectors, most of which are non-neighbors. This paper presents ANNS-AMP, an adaptive mixed-precision framework and accelerator that adapts the precision of distance computation to the characteristics of queries and data distribution. The key insight is that different regions of the vector space require different levels of precision to preserve top-k accuracy. ANNS-AMP leverages the clustered structure of PQ-based indices and introduces a lightweight predictor to determine cluster-level precision at runtime. According to our experiments on representative datasets, ANNS-AMP achieves 163.76x, 10.57x, and 2.06x performance speedups on average, and reduces average energy consumption by 1100.00x, 39.41x, and 6.66x compared to CPU, GPU, and customized ANNS accelerator baselines, respectively, while maintaining accuracy loss below 2.7%.

**中文翻译:** 近似最近邻搜索（ANNS）是 LLM 和推荐系统等现代应用中的关键内核。然而，其效率根本上受限于需要在查询和大量高维向量之间计算距离（其中大多数不是邻居）。本文提出 ANNS-AMP，一种自适应混合精度框架和加速器，可根据查询和数据分布的特征自适应调整距离计算的精度。关键洞察在于向量空间的不同区域需要不同精度水平来保持 top-k 准确性。ANNS-AMP 利用基于 PQ 的索引的聚类结构，并引入轻量级预测器在运行时确定聚类级别精度。在代表性数据集上的实验表明，与 CPU、GPU 和定制 ANNS 加速器基线相比，ANNS-AMP 分别实现平均 163.76 倍、10.57 倍和 2.06 倍的性能加速，并将平均能耗降低 1100.00 倍、39.41 倍和 6.66 倍，同时精度损失低于 2.7%。
