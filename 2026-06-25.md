### 1. Constraint Tax in Open-Weight LLMs: Tool Calling Suppression Under Structured Output
- **链接**: https://arxiv.org/abs/2606.25605v1
- **作者**: Fangzheng Li, Aimin Zhang, Chen Lv
- **关键词**: Structured Output, Tool Calling, Agent System, Inference-time Strategy

**Abstract**:
Tool Calling and Structured Output are two core capabilities of modern Agent systems, yet their interaction under joint deployment conditions remains insufficiently understood. This paper reports a reproducible phenomenon observed in a production Agent system: when Tool Calling and JSON Schema constraints are simultaneously enabled, multiple open-weight models cease invoking tools despite maintaining high schema compliance. We refer to this behavior as Tool Suppression. Through controlled experiments across multiple model families and deployment settings, we consistently reproduce Tool Suppression under joint constraints, while tool execution and schema compliance remain functional when evaluated independently. Further analysis reveals that JSON Schema constraints are compiled into grammar-based token masks, causing tool-call tokens to become unreachable during decoding. This provides an implementation-level explanation for the observed behavior. To mitigate the problem, we propose Transparent Two-Pass Execution, an inference-time strategy that decouples tool execution from schema-constrained response generation.

**中文摘要**：
Tool Calling 和结构化输出是现代 Agent 系统的两个核心能力，但它们在联合部署条件下的相互作用尚未被充分理解。本文报告了一个在生产 Agent 系统中观察到的可复现现象：当同时启用 Tool Calling 和 JSON Schema 约束时，多个开放权重模型停止调用工具，尽管仍保持高度的 Schema 合规性。作者将这种行为称为 Tool Suppression。进一步分析表明，JSON Schema 约束被编译为基于语法的 token mask，导致 tool-call token 在解码过程中变得不可访问。为缓解该问题，作者提出了 Transparent Two-Pass Execution（透明两遍执行），一种将工具执行与 Schema 约束响应生成解耦的推理时策略，无需模型重新训练即可恢复工具调用。

## 🚀 推理加速

### 2. Quantization Inflates Reasoning: Token Inflation as a Hidden Cost of Low-Bit Reasoning Models
- **链接**: https://arxiv.org/abs/2606.25519v1
- **作者**: Xinyu Lian, Walid Krichene, Beichen Huang, Masahiro Tanaka, Olatunji Ruwase, Li Zhang, Minjia Zhang (Microsoft)
- **关键词**: Quantization, Reasoning Model, Inference Cost, CoT Token Inflation, Serving

**Abstract**:
Quantization is widely used to reduce the inference cost of large language models, but its effect on reasoning models is not fully captured by final-answer accuracy or per-token latency. We show that low-bit post-training quantization can introduce a hidden test-time compute cost: quantized reasoning models often generate longer chains of thought even when they still answer correctly. Across mathematical reasoning, code generation, scientific question answering, and agentic tool-use benchmarks, we find that INT4/INT3 quantization can preserve accuracy but increase reasoning-token usage, offsetting the expected per-token speedup. To measure this effect, we introduce the CoT Token Inflation Ratio, which compares reasoning length between quantized and full-precision models averaged across all evaluation benchmarks. We further show that token inflation is accompanied by behavioral changes in the reasoning trace, including more intermediate steps and greater semantic repetition. These changes translate into measurable end-to-end real-world serving penalties. Finally, we evaluate mitigation strategies and find that prompting and decoding-time sampling offer inconsistent accuracy-length trade-offs, while quantization-aware training shows more promise in reducing both accuracy degradation and token inflation.

**中文摘要**：
量化被广泛用于降低大语言模型的推理成本，但其对推理模型的影响不能仅通过最终答案准确性或每 token 延迟来衡量。本文发现低比特后训练量化会引入隐藏的测试时计算成本：量化后的推理模型即使在正确回答问题时，也往往会生成更长的思维链。在数学推理、代码生成、科学问答和 Agent 工具使用等基准测试中，INT4/INT3 量化可以保持准确性，但会增加推理 token 的使用量，抵消了预期的每 token 加速。作者提出了 CoT Token 膨胀率来衡量这种效应。进一步发现 token 膨胀伴随推理轨迹的行为变化，包括更多的中间步骤和更多的语义重复。这些变化转化为可测量的端到端真实服务惩罚。评估缓解策略发现，提示词和解码时采样的效果不一致，而量化感知训练在减少准确性下降和 token 膨胀方面更有前景。这一结果建议在评估量化推理模型时，应将推理 token 使用量作为与准确性并重的指标报告。

### 3. Agentic Evolution of Physically Constrained Foundation Models
- **链接**: https://arxiv.org/abs/2606.25532v1
- **作者**: Jiangwei Zhang, Wen Sun, Chong Wang, Shiyao Li, Cheng Che, Chunjing Han, Dan Meng, Jian Yang, Yu Wang, Rui Hou
- **关键词**: Model Compression, Quantization, MoE, Hardware-Software Co-Design, Deployment

**Abstract**:
Artificial intelligence increasingly drives automated scientific discovery, yet contemporary generalist agents lack physical grounding, frequently hallucinating hardware-incompatible designs. Here, we present a physically grounded, multi-agent discovery engine that autonomously architects hardware-compliant computing systems. Applied to the extreme testbed of foundation model deployment, the engine evolved two hardware-aware compression methodologies surpassing human-engineered heuristics: Q-Enhance mitigates long-context accuracy loss in dense models, and MoE-Salient-AQ outperforms state-of-the-art manual sparse Mixture-of-Experts designs by 3.7% at sub-3-bit regimes. Utilizing a bandwidth-efficient Sensitivity Profile, we successfully deployed a massive 235-billion-parameter model onto a constrained dual-A100 server, reducing memory requirements by 75% with a marginal 0.64% accuracy degradation.

**中文摘要**：
本文提出了一个基于物理感知的多智能体自动发现引擎，可自主设计硬件兼容的计算系统。在基础模型部署这一极端测试场景中，该引擎演化了两种超越人工设计的硬件感知压缩方法：Q-Enhance 缓解了密集模型的长上下文精度损失，MoE-Salient-AQ 在 sub-3-bit 量化下超越最先进的手工稀疏 MoE 设计 3.7%。利用带宽高效的 Sensitivity Profile，作者成功将 2350 亿参数的巨型模型部署到受限的双 A100 服务器上，将内存需求降低 75%，准确率仅下降 0.64%。

### 4. HiReLC: Hierarchical Reinforcement Learning for Neural Network Compression — Pruning and Quantization
- **链接**: https://arxiv.org/abs/2606.26002v1
- **作者**: Kamar Hibatallah Baghdadi, Kawther Guoual Belhamidi, Sara Belhadj, Aissa Boulmerka, Nadir Farhi
- **关键词**: Neural Network Compression, Quantization, Pruning, RL-based Optimization

**Abstract**:
We present HiReLC, a hierarchical ensemble-reinforcement learning framework for automated joint quantization and structured pruning of deep neural networks. The framework decomposes the compression search across two levels of abstraction: low-level agents (LLAs) operate independently per block, selecting per-kernel configurations over a multi-discrete action space spanning bitwidth, pruning keep-ratio, quantization type, and granularity, while high-level agents (HLAs) coordinate global budget allocation via ensemble voting guided by Fisher Information-based sensitivity estimates. Experiments across Vision Transformer and CNN benchmarks demonstrate effective parameter-storage compression ratios of 5.99-6.72× with minimal accuracy degradation.

**中文摘要**：
HiReLC 是一个分层集成强化学习框架，用于深度神经网络的自动联合量化与结构化剪枝。该框架将压缩搜索分解为两个抽象层次：低层智能体独立按块操作，在多离散动作空间中选择逐核配置（位宽、剪枝保留率、量化类型和粒度）；高层智能体通过基于 Fisher Information 敏感性估计的集成投票协调全局预算分配。在 Vision Transformer 和 CNN 基准测试上的实验表明，该方法实现了 5.99-6.72 倍的参数存储压缩比，准确率损失极小。

### 5. BitNet Text Embeddings (BITEMBED)
- **链接**: https://arxiv.org/abs/2606.25674v1
- **作者**: Zhen Li, Xin Huang, Liang Wang, Nan Yang, Ting Song, Yan Xia, Xun Wu, Shaohan Huang, Huishuai Zhang, Furu Wei, Dongyan Zhao
- **关键词**: Quantization, Embedding, BitNet, Inference Efficiency, Storage

**Abstract**:
LLM-based text embedders have substantially improved retrieval and semantic representation quality, but their deployment remains costly: large backbone models slow down embedding inference, while high-dimensional full-precision embeddings impose substantial storage and bandwidth overhead on large-scale indexes. In this paper, we present BITEMBED, an extreme low-bit framework for LLM-based text embedding that jointly targets encoding efficiency and vector storage. BITEMBED converts pretrained LLM backbones into BitNet-style embedding encoders with ternary weights, quantized activations, and lightweight normalization refinement. Experiments on MMTEB (eng, v2) with Qwen3-0.6B and Gemma3-270M show that BITEMBED is largely comparable to full precision teacher embedders.

**中文摘要**：
基于 LLM 的文本嵌入模型显著提升了检索和语义表示质量，但其部署成本仍然高昂：大骨干模型降低了嵌入推理速度，高维全精度嵌入给大规模索引带来了巨大的存储和带宽开销。本文提出 BITEMBED，一个面向 LLM 文本嵌入的极端低比特框架，同时针对编码效率和向量存储进行优化。BITEMBED 将预训练 LLM 骨干转换为 BitNet 风格的嵌入编码器，使用三值权重、量化激活和轻量级归一化精炼。在 Qwen3-0.6B 和 Gemma3-270M 上的 MMTEB 基准测试中，BITEMBED 的性能与全精度教师嵌入模型基本相当。

### 6. NEURON-Fabric: Architecture-Runtime Co-Design for Controlled Low-Bit Gradient Communication
- **链接**: https://arxiv.org/abs/2606.25759v1
- **作者**: Ziqiang Wang, Changcheng Huang, Chung-Horng Lung
- **关键词**: Distributed Training, Gradient Communication, Low-Bit Aggregation, Runtime System

**Abstract**:
Large-scale neural-network training repeatedly aggregates gradients across devices, making communication a central cost in distributed learning. Low-bit gradient aggregation can reduce this cost, but applying it as a static replacement for full-precision communication can destabilize training because safe precision depends on training phase, model structure, runtime bucketization, and the communication substrate. This paper presents NEURON-Fabric, a profile-guided runtime system for controlled low-bit gradient communication. NEURON-Fabric uses calibrated operating profiles, model-aware runtime bindings, online training-health monitoring, and reducer-capacity checks to decide when low-bit aggregation should be admitted, when execution should fall back to FP32, and which model regions are eligible for each route. The runtime preserves model semantics inside mixed DDP buckets and treats reducer admission as an architecture-runtime co-design problem rather than as a standalone compression operator. Across vision, Transformer, and autoregressive language-model workloads, NEURON-Fabric validates the path from calibration to distributed communication-hook execution. Static low-bit communication can collapse training accuracy, while profile-guided control preserves accuracy near full-precision references.

**中文摘要**：
大规模神经网络训练需要反复跨设备聚合梯度，使通信成为分布式学习的核心瓶颈。低比特梯度聚合可以降低通信成本，但将其作为全精度通信的静态替代会破坏训练稳定性。本文提出 NEURON-Fabric，一个基于 profile 引导的运行时系统，用于受控的低比特梯度通信。NEURON-Fabric 使用校准操作 profile、模型感知的运行时绑定、在线训练健康监控和 reducer 容量检查，来决定何时允许低比特聚合、何时回退到 FP32、以及哪些模型区域适合每种路径。在视觉、Transformer 和自回归语言模型任务上的实验表明，静态低比特通信可能导致训练精度崩溃，而 profile 引导的控制则能将精度保持在接近全精度参考的水平。
